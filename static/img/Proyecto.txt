***GIT CMD

Ve a https://git-scm.com/download/win y descarga el instalador.

Ejecuta el instalador con las opciones por defecto.

Asegúrate de marcar “Git from the command line and also from 3rd-party software” para que puedas usarlo en PowerShell.

Termina la instalación.

2️⃣ Verificar la instalación

Abre una nueva terminal (PowerShell) y escribe:

git --version


Deberías ver algo como:

git version 2.42.0.windows.1

3️⃣ Configurar Git (solo la primera vez)
git config --global user.name "Tu Nombre"
git config --global user.email "tuemail@example.com"


Esto es necesario para que Git registre tus commits correctamente.

1️⃣ Crea una carpeta para tu proyecto

Por ejemplo:

C:\Users\guido.cedeno\Documents\Tayron\


Ahí debes tener tu proyecto con:

app.py
templates/
static/
requirements.txt
Procfile

2️⃣ Abre PowerShell en esa carpeta
cd C:\Users\guido.cedeno\Documents\Tayron

3️⃣ Inicializa Git solo aquí
git init
git add .
git commit -m "Primer commit del proyecto Churros de Colores"


✅ Esto evita que Git intente rastrear archivos del sistema.

4️⃣ Crear y conectar a GitHub

Crea un repositorio en GitHub llamado churros-de-colores.

Conecta tu repo local:

git branch -M main
git remote add origin https://github.com/xdogui/churroland
git push -u origin main

🔹 Pasos para desplegar tu app Flask en Fly.io
1. Instalar la CLI de Fly.io

En tu máquina (Windows PowerShell):

iwr https://fly.io/install.ps1 -useb | iex


👉 Esto instala el comando flyctl.

Verifica que funcione:

flyctl --version

2. Loguearte en Fly.io

Ejecuta:

flyctl auth login

3. Inicializar el proyecto

Dentro de la carpeta de tu proyecto (C:\Users\guido.cedeno\Downloads\Tayron):

flyctl launch


Te preguntará:

App Name → puedes poner churroland (o deja que genere uno).

Region → escoge la más cercana (ej: mia = Miami, scl = Santiago).

Database → di que no (porque tu app no usa DB todavía).

Esto crea un archivo fly.toml.
